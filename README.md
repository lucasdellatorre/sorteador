# Execute o backend
Vá para o backend
```
cd src/backend
```
Suba a aplicação com docker
```
docker compose up --build
```

# Execute o frontend
Vá para o front
```
cd src/frontend
```
Instale as dependências com npm
```
npm  i
```
Execute a aplicação
```
npm run dev // iniciar aplicacao na porta 3000
```
